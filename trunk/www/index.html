Contains RI, TCK, examples, tools for JAIN-SIP-1.1 (JSR-32) and an implementation of JAIN-SDP (JSR-141). JAIN-SIP RI is a full implementation of RFC 3261.
Tools include a signaling trace viewer that can take input from an etherreal/stack trace.
RI, TCK and tools are in the public domain. See the SIP Communicator Project on java.net for a soft phone built on JAIN-SIP. See jain-sip-presence-proxy for a proxy server and IM client using JAIN-SIP.
