Contains RI, TCK, examples, tools for JAIN-SIP-1.1 (JSR-32 maintenance release) and an SDP library that conforms to the public release of JSR 141 (JAIN-SDP) interfaces. JAIN-SIP RI is a full implementation of RFC 3261. JAIN-SDP interfaces are still under public review and are subject to change.
Tools include a signaling trace viewer that can take input from an etherreal/stack trace.
For the benifit of those doing SIP-related research, the stack can be built either as a real stack (using the network) or as a discrete event simulation to allow you to simulate distributed SIP applications. Contact mranga@dev.java.net for the simulation code details.
RI, TCK and tools are in the public domain. See JAIN-SIP-PRESENCE-PROXY for a proxy server and IM client using JAIN-SIP. See the SIP Communicator Project on java.net for a soft phone built on JAIN-SIP.
Vist the NIST-SIP project at the NIST Advanced Networking Technologies Division and join our mailing list.
