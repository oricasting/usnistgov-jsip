Change-Log for JSIP v1.2

Authors: Phelim O'Doherty (BEA) M. Ranganathan (NIST)

Package javax.sip

  • Add new class IOExceptionEvent.java
  • Add new class DialogTerminatedEvent.java
  • Add new class TransactionTerminatedEvent.java
  • Remove the 1-1 mapping from SipProvider to listener. There is now a 1-many mapping from SipListener to SipProvider. Each SipListener of a given SipProvider has the same IP Address and Port but differs in transport.
